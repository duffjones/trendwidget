Google Sheets to Highcharts Integration

What We Do

We use the Google Sheets API to save data, and serve it to the web where it can be displayed with Highcharts. As an example of what can be done with this pipeline, we've created some weather charts and descriptions below. Just with data collected on our phones, and stored on google drive, we can create endless tables, visualizations and injections that allow you to share your data live in creative, interesting ways.
